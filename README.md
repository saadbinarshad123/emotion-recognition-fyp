# Text-Based Emotion Recognition System

## Overview

This project is a Final Year Project that detects human emotions from text using Natural Language Processing (NLP) and Deep Learning.

It uses a pretrained ALBERT model and can classify text into 13 different emotions with around 80% accuracy.

The application includes a simple desktop interface built with Tkinter, making it easy to use without any complex setup.

---

## Features

* Emotion prediction from text input
* 13 emotion categories
* Lightweight desktop interface
* Works on CPU (no GPU required)

---

## Technologies Used

* Python
* PyTorch
* Transformers (HuggingFace)
* Scikit-learn
* Pandas
* Tkinter

---

* CODE.py → main application
